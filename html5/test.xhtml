HTML5 <input type="email"> validation tests

This XHTML5 file serves to test how browsers implement HTML5 <input type="email"> validation, and thus how the pg_html5_email_address PostgreSQL extension should handle this validation to be in sync with both the HTML5 spec and the implementations thereof.
